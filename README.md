# 🚀 Déploiement professionnel

### Lancer l’application
\`\`\`bash
pm2 start ecosystem.config.js
pm2 save
\`\`\`

### Relancer après redémarrage
\`\`\`bash
pm2 resurrect
\`\`\`

### Vérifier l’état
\`\`\`bash
pm2 list
\`\`\`
